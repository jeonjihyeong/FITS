현재 상태
-FRONT 
    vue 사용해서 기본적인 레이아웃 정도 구현해놓은 상태 카카오 로그인이나 그런거에서 상세함은 없음

-SERVER
    node.js 사용해서 공부하는 중 현재 회원관리, 게시판 기능 정도만 가능한 웹사이트 하나 가지고 있음 + 여기서 추가해야될 기능 생각해보기
    nest.js 에서는 기본 express 기반 서버를 기준으로 똑같이 기능할 수 있는 서버를 만들어볼 예정 / 현재 로그인 + 회원가입 (But 토큰, 메일 같은 기능 없음)

-DevOps
    docker로 배포를 할려고 공부중 하지만 레디스에서 오류가 자꾸 나옴/ 중복로그인 방지 등을 해보기 위한 시도
